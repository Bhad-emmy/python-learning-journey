DAY 8 – PYTHON LISTS

Topics Covered

1. Creating Lists
A list stores multiple values in one variable.

Example:
fruits = ["Apple", "Orange", "Banana"]

2. Printing Lists

print(fruits)

Output:
['Apple', 'Orange', 'Banana']

3. Indexing

Python starts counting from 0.

fruits[0] = Apple
fruits[1] = Orange
fruits[2] = Banana

4. Modifying Items

fruits[1] = "Mango"

5. Adding Items

append()

Example:
fruits.append("Pineapple")

6. Removing Items

remove()

Example:
fruits.remove("Mango")

7. Counting Items

len()

Example:
print(len(fruits))

8. Looping Through Lists

for fruit in fruits:
    print(fruit)

Python takes one item at a time from the list until it reaches the end.

Example Output:

Apple
Orange
Banana
Mango

Loop Example

scores = [50, 70, 90]

for score in scores:
    print(score + 10)

Output:

60
80
100

Key Lessons

✓ Lists store multiple values.
✓ Python starts indexing from 0.
✓ append() adds items.
✓ remove() removes items.
✓ len() counts items.
✓ for loops process every item in a list automatically.
